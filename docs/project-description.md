#FixNow - תיאור פרויקט

FixNow היא מערכת לחיבור בין לקוחות שצריכים לבין בעלי מקצוע זמינים

##משתמשי המערכת
-בעל מקצוע
-לקוח

##פעולות עיקריות
-הרשמה והתחברות
-פתיחת קריאת שירות
-צפייה בקריאות פתוחות
-קבלת קריאה על ידי בעל המקצוע
-שינוי סטטוס קריאה
-דירוג בעל מקצוע

##סוגי בעלי מקצוע
-חשמלאי
-אינסטלטור
-טכנאי מזגנים
-מנעולן
-צבעי
