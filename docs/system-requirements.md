# דרישות מערכת - FixNow

## מטרת המערכת
מערכת FixNow נועדה לחבר בין לקוחות שצריכים שירות לבין בעלי מקצוע זמינים.

## משתמשי המערכת

### לקוח
- הרשמה למערכת
- התחברות
- פתיחת קריאת שירות
- צפייה בקריאות שפתח
- דירוג בעל מקצוע לאחר סיום עבודה

### בעל מקצוע
- הרשמה למערכת
- בחירת תחום מקצוע
- צפייה בקריאות פתוחות
- קבלת קריאה
- עדכון סטטוס עבודה

## דרישות פונקציונליות
- המערכת תאפשר הרשמה והתחברות.
- המערכת תאפשר ללקוח לפתוח קריאת שירות.
- המערכת תציג לבעלי מקצוע קריאות רלוונטיות.
- בעל מקצוע יוכל לקבל קריאה.
- לקוח יוכל לראות את סטטוס הקריאה שלו.
- לקוח יוכל לדרג בעל מקצוע.

## דרישות לא פונקציונליות
- המערכת תהיה פשוטה ונוחה לשימוש.
- המערכת תתמוך בעברית.
- המערכת תעבוד בדפדפן.
- המערכת תשמור נתונים במסד נתונים.
