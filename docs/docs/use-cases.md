# Use Cases - FixNow

## פתיחת קריאת שירות

שחקן: לקוח

תיאור:
הלקוח נכנס למערכת, בוחר סוג בעל מקצוע, ממלא תיאור תקלה, כתובת וטלפון ולוחץ על "שלח קריאה".

תוצאה:
הקריאה נשמרת במערכת וממתינה לבעל מקצוע.

---

## קבלת קריאה

שחקן: בעל מקצוע

תיאור:
בעל המקצוע רואה רשימת קריאות פתוחות ולוחץ על "קבל קריאה".

תוצאה:
הקריאה משויכת אליו.

---

## התחברות

שחקן: כל משתמש

תיאור:
המשתמש מזין אימייל וסיסמה.

תוצאה:
המערכת מאמתת את הנתונים ומכניסה אותו למערכת.
